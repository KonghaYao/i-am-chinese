# 信息验证任务

使用 web search 能力验证 skill 文件中的所有信息。

## 执行流程

1. **规划验证内容**

    - 读取 skill 文件，提取所有需要验证的关键信息
    - 在 `skills/[skill_name]/validate.md` 中创建验证清单（使用 `[ ]` 待办格式）
    - 按主题分节：事实性陈述、数据统计、引用来源等

2. **并发验证**

    - 同时发起多个搜索请求，逐一验证各章节
    - 验证通过则标记 `[x]`
    - 验证失败则保留 `[ ]` 并追加说明

3. **记录结果** 每项验证后记录：
    ```
    - [ ] {{原内容}}
      ❌ 验证失败
      📝 问题：{{具体错误}}
      🔗 来源：{{参考网址}}
    ```

## 输出要求

-   所有陈述需有可查证来源
-   优先使用权威网站（官网、学术、官方统计）
-   不确定信息标注为"待进一步核实"
