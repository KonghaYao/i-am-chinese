# 客家建筑文化

客家建筑是中国传统建筑的重要组成部分，体现了客家人的智慧、团结和生存智慧。

## 建筑类型

### 土楼

**概述**
- 世界文化遗产（2008年入选）
- 主要分布在福建永定、南靖等地
- 圆形或方形建筑
- 可住几十户甚至上百户人

**特点**
- 防御性强
- 团聚性好
- 冬暖夏凉
- 节省材料

**圆形土楼**
- 最具代表性
- 布局合理
- 采光通风好
- 象征团结

**方形土楼**
- 早期形式
- 也有大型方形土楼
- 布局规整
- 防御功能强

**著名土楼**
- 振成楼（永定）
- 承启楼（永定）
- 田螺坑土楼群（南靖）
- 河坑土楼群（南靖）

### 围龙屋

**概述**
- 广东、江西客家特色民居
- 分布在梅州、赣州等地
- 半圆形布局
- 前半部为半圆形

**布局特点**
- 前半部：半月形池塘
- 中部：堂屋（祠堂）
- 后半部：半圆形房舍
- 整体呈椭圆形

**功能**
- 居住功能
- 防御功能
- 祭祀功能
- 社交功能

**典型代表**
- 梅州围龙屋
- 赣州围屋
- 龙南围屋

### 四角楼

**特点**
- 四角有炮楼
- 防御性强
- 方形布局
- 高墙深院

**分布**
- 广东东部
- 江西南部
- 福建西部

**功能**
- 防御盗匪
- 保护族人
- 展示财富
- 社会地位
